# Password Generator

Random password generator that creates secure passwords from 8 to 50 characters. Built with HTML, CSS, and JavaScript.

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**

## Features

- Generate passwords from 8-50 characters
- Customizable character sets
- Copy to clipboard
- Password strength indicator

## Getting Started

1. Clone the repository
2. Open \index.html\ in your browser
